Calculator App

Project Description

The Calculator App is a simple web application used to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

Features

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • User Friendly Interface
  • Responsive Design

Technologies Used

  • HTML5
  • CSS3
  • JavaScript

Installation

  1. Download the project.
  2. Open the project folder.
  3. Run index.html.

Usage

  1. Enter two numbers.
  2. Select an operation.
  3. See the result.
